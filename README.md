# 실행방법
1. redis실행
2. git클론
3. cd 클론받은 디렉토리 경로
4. 빌드
2. .jar실행

```text
redis-server
git clone
cd [디렉토리 경로]
./gradlew clean build
cd build/libs
java -jar _11st-0.0.1-SNAPSHOT.jar

```
